# dstest-components 组件库

### 快速开始

#### 1.安装组件库

```bash
npm i h-components
```

#### 2.引入组件库

```javascript
//全部引入
import "h-components/dist/css/index.css";
import H from "h-components";
Vue.use(H);

//按需引入
import "h-components/dist/css/demo.css";
import { Demo } from "h-components";
Vue.use(Demo);
```
